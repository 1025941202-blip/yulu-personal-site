公开仓库 / Codex Skill

Ian 小黑怪诞正文配图

这个 Skill 用来把中文文章里的判断、流程、结构和隐喻，转成 16:9 白底手绘正文配图。它的核心不是“随便配一张图”，而是先找到文章里的认知锚点，再让小黑这个黑色小角色参与到概念动作里。

适合 中文长文、方法论内容、AI 工作流说明、抽象概念解释图。
网址 github.com/helloianneo/ian-xiaohei-illustrations
安装 复制仓库里的 ian-xiaohei-illustrations/ 子目录到本机 ~/.codex/skills/，然后重启 Codex。 
Use $ian-xiaohei-illustrations 为这篇中文文章设计 5 张小黑正文配图。

Use $ian-xiaohei-illustrations 为“信任不是喊出来的，而是一块证据一块证据铺过去”生成一张解释图。

适合 真人 IP、生图参考、数字人视频、课程宣传和个人品牌素材库。

Use $ip-character-photo-kit 看看这组照片能不能做稳定的 IP 形象资产包。

适合 AI 视频生成、IP 故事、课程讲解、工具演示和活动宣传。
